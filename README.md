# 工具使用說明（須使用 Linux 環境）
1. 取得專案
```shell
git clone https://github.com/flyingsnake5254/Apk_Fuzzer.git
```
2. 將要測試的 apk 放到 Apk_Fuzzer/apks 資料夾下
3. 執行 apkfuzzer.sh 腳本
```shell
sh apkfuzzer.sh --auto
```
4. 檢視 reports 資料夾下的測試報告
